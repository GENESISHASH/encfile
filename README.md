# install

using [npm](https://npmjs.org)

```
npm i encfile --save
```

# example

``` coffeescript
encfile = require 'encfile'
```

# shell example

``` shell
encfile e --infile file.zip --outfile file.enc --key secret
encfile d --infile file.zip.enc --outfile file.zip.enc --key secret
```
